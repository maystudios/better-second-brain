# Second Brain Pattern (Map of Content)

This is the navigation hub for the bootstrap domain: how to use an LLM to build and maintain a personal knowledge base, and how a Better Second Brain (BSB) extends that idea. Start with the canonical pattern, then read the concepts, sources, and entities that make up the stack. For the build-time view of how BSB is assembled, see [[wiki/moc/bsb-architecture]].
