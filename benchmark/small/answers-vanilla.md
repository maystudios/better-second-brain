# Answers — uv (vanilla wiki)

1. uv is written in **Rust**, and Astral describes it as "an extremely fast Python package and project manager." (Source: index.md / github.com/astral-sh/uv, docs.astral.sh/uv)

2. The exact Windows standalone installer command shown is:
   `powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"` (Source: installation.md)

3. `uvx` is documented as an exact alias for (exactly equivalent to) **`uv tool run`**. (Source: tools.md)

4. uv aims to replace `pip`, `pip-tools`, `pipx`, `poetry`, `pyenv`, `twine`, and `virtualenv`, and is claimed to be **10–100x faster than pip**. (Source: index.md / github.com/astral-sh/uv)

5. The pinned/specific version used in the installation example is **0.11.22** (`https://astral.sh/uv/0.11.22/install.sh`). (Source: installation.md)
