## Key claims

- uv is described as "an extremely fast Python package and project manager, written in Rust."
- uv replaces multiple tools, named explicitly: pip, pip-tools, pipx, poetry, pyenv, twine, and virtualenv.
- Performance is "claimed to be 10-100x faster than pip."
- Features include project management with universal lockfiles, script execution with inline dependency declarations, Python version installation/management, and tool execution similar to pipx.
- Supports Cargo-style workspaces and maintains a global cache for efficient disk usage.
- Installation (macOS/Linux): `curl -LsSf https://astral.sh/uv/install.sh | sh`
- Installation (Windows): `powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"`
- Installation via PyPI: `pip install uv` or `pipx install uv`
- Self-updates: `uv self update`
- Core feature areas: Projects, Scripts, Tools, Python Management, Pip Interface, Workspace Support, Cross-Platform (macOS, Linux, Windows).
- uv is developed by Astral, "creators of Ruff and other Python tooling."
