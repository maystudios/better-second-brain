# modelcontextprotocol/modelcontextprotocol (GitHub)

Source: https://github.com/modelcontextprotocol/modelcontextprotocol

This GitHub repository hosts the specification and documentation for the Model Context Protocol, an open standard for connecting AI models with external tools and context.

## What this repo contains

1. **MCP Specification** - the formal protocol definition
2. **Protocol Schema** - available in both TypeScript and JSON Schema formats
3. **Official Documentation** - built with Mintlify, hosted at modelcontextprotocol.io

## Key details

- **Latest Version:** 2025-11-25 (released November 25, 2025)
- **Schema Definition:** The schema is defined in TypeScript first, but made available as JSON Schema as well, for wider compatibility.
- **Primary Language:** TypeScript (~84% of codebase)
