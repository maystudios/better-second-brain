## Key claims
- This repository hosts the specification and documentation for MCP, an open standard for connecting AI models with external tools and context.
- Repo contains: the MCP Specification (formal protocol definition), the Protocol Schema (in both TypeScript and JSON Schema formats), and the Official Documentation (built with Mintlify, hosted at modelcontextprotocol.io).
- Latest version: 2025-11-25 (released November 25, 2025).
- The schema is defined in TypeScript first, then made available as JSON Schema for wider compatibility.
- Primary language: TypeScript (~84% of codebase).
- Authors: David Soria Parra (@dsp) and Justin Spahr-Summers (@jspahrsummers).
- License: MIT.
- Complete documentation is available at modelcontextprotocol.io; contributions guided by CONTRIBUTING.md.
