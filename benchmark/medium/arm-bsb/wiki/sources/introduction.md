## Key claims
- MCP (Model Context Protocol) is an open-source standard for connecting AI applications to external systems.
- Using MCP, AI applications like Claude or ChatGPT can connect to data sources (e.g. local files, databases), tools (e.g. search engines, calculators), and workflows (e.g. specialized prompts).
- MCP is analogized to "a USB-C port for AI applications" — a standardized way to connect AI applications to external systems.
- Example enablements stated: agents accessing Google Calendar and Notion; Claude Code generating a web app from a Figma design; enterprise chatbots connecting to multiple databases; AI models creating 3D designs on Blender and printing via a 3D printer.
- Benefits by ecosystem role: developers (reduced development time/complexity), AI applications/agents (access to ecosystem of data sources, tools, apps), end-users (more capable AI applications/agents).
- MCP is an open protocol supported across a wide range of clients and servers. Named supporting tools: Claude, ChatGPT, Visual Studio Code, Cursor, MCPJam, and "many others."
- Stated build paths: build servers, build clients, build MCP Apps (interactive apps that run inside AI clients).
